# main-mk-journal

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态文件的形式存放，不针对任何特定域名或网站，仅作为文档与内容的存储与展示空间。

## 项目简介

`main-mk-journal` 是一个以 HTML 页面为主要内容的归档仓库。仓库中的页面彼此独立，内容涵盖记录、参考、整理等类型。适合需要长期保存、易于浏览的静态页面集合。

## 目录说明

仓库的根目录下按页面或主题放置对应的 HTML 文件及附属资源（如图片、样式、脚本等）。每个页面通常位于独立的子目录中，便于维护与引用。

```
/
├── page-a/          # 页面 A 相关文件
│   ├── index.html
│   └── assets/
├── page-b/          # 页面 B 相关文件
│   ├── index.html
│   └── assets/
└── README.md        # 本文件
```

目录结构可能随内容增加而调整，建议关注根目录下的更新。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面之间无强制关联，可单独访问。
- 内容以归档和记录为目的，不涉及商业推广或定向引导。
- 页面可能包含外部资源（如字体、图片），但尽量保持依赖最小化。

## 维护说明

- 本仓库由维护者不定期更新。
- 如需提交新页面或修改现有内容，欢迎发起 Pull Request。
- 提交前请确保页面结构清晰、资源路径正确。
- 不建议在页面中添加跟踪代码、广告或外部链接跳转。
